About the Project

An end-to-end applied machine learning project — from raw data to a deployed, production-styled banking platform.

Problem Statement

Manual credit card approval review is slow, inconsistent and hard to scale. This project builds a machine learning system that predicts whether an applicant is likely to be approved, based on their demographic and financial profile.

Objectives

  • Build an accurate, explainable approval prediction model
  • Wrap the model in a premium, responsive banking-style UI
  • Provide real analytics: dashboard, history visibility
  • Make the app deployable end-to-end on Render

Dataset

The Credit Card Approval Prediction dataset (Kaggle), combining two sources:

application_record.csv438,557 applicant records
credit_record.csv1,048,575 monthly credit records

The two sources are inner-joined on applicant ID and labeled using each applicant's worst historical credit status, then deduplicated - yielding 11,106 unique applicant profiles across 13 engineered features.
