Project Overview

CreditSense is a Flask-based web application that predicts credit card approval likelihood using a trained Random Forest model. It includes a multi-step prediction wizard, a live analytics dashboard, prediction history, and an ML model comparison page. This guide walks you through running the full project on your own machine.

System Requirements

Python Version3.11.x (recommended) or 3.10+
Package Managerpip (bundled with Python)
Operating SystemWindows, macOS, or Linux
Required PackagesFlask, scikit-learn, pandas, numpy, gunicorn, fpdf2

All required packages are pinned in requirements.txt — you don't need to install them individually.

Step-by-Step Local Setup

1

Clone the repository

git clone https://github.com/JohnsonpaulG/Credit-Card-Approval-Prediction.git
cd Credit-Card-Approval-Prediction
2

Create a virtual environment

# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
3

Install dependencies

pip install -r requirements.txt
4

Run the Flask application

python app.py

This starts Flask's built-in development server on port 5000.

5

Open the app in your browser

Navigate to the default URL below. The app opens directly to the Home page.

Default URLhttp://localhost:5000
